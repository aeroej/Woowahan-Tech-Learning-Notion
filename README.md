# Woowahan-Tech-Learning-Notion   
[우아한테크러닝4기 - 나만의 노션만들기]()에서 실습한 내용을 정리한 리포지토리입니다.   
## 기술
### Front-end
- React
### Back-end
- Express
- MariaDB
### Language
- Typescript   
## 리포지토리 실행
1. 리포지토리를 클론합니다.
```bash
git clone https://github.com/aeroej/Woowahan-Tech-Learning-Notion.git
```
2. 백엔드 모듈 설치 후 실행합니다.   
```bash
cd back-end
yarn install
yarn serve
```
3. 프런트엔드 모듈 설치 후 실행합니다.    
```bash
cd ../front-end
yarn install
yarn serve
```
